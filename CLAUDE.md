# ciro 事業の憲法

## 屋号・発信者
- 屋号: ciro
- 発信者: 赤川知佳

## 想定読者
- フリーランス・小規模チームのデザイナー／クリエイター
- AIに興味はあるが、情報過多で何から触ればいいか分からない層
- 「仕事を奪われる側」ではなく「使いこなす側」に回りたい人

## note の目的
1. AIニュースを現場視点で翻訳する（技術解説ではなく「あなたの仕事がどう変わるか」）
2. ciro の信頼資産を積む（実装者としての視点）
3. 単発記事ではなく連続読者を作る

## 文体ルール
- 一人称: 「私」
- 敬体（です・ます）基本、冒頭・要所は短文で断言
- 専門用語は日常語へ言い換え or 一言で補足
- 箇条書きは3〜5項目まで
- 1記事 1,500〜2,500字目安
- 見出し H2 中心、H3は最小限
- リード文は「これを読むと何が分かるか」を2〜3行で明記
- 末尾は CTA（ciro のサービス／次回予告／感想募集いずれか）

## NGワード・禁則
- 煽り表現（「絶対」「必ず」「〇〇はオワコン」など）
- AI を擬人化して万能扱いしない（「AIが考えてくれる」→ 「AIが下書きを出す」）
- 根拠のない数値（「生産性〇倍」など）禁止。出典必須。
- 他社サービスを名指しで貶さない

## 品質基準（毎記事チェック）
- [ ] リード文で価値が一目で分かる
- [ ] 固有名詞・日付・数値に出典あり
- [ ] 読者の「次の行動」が明確
- [ ] 誤字脱字ゼロ
- [ ] タイトルは32字以内、数字または具体名を含む
