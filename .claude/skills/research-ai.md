# research-ai

## 目的
毎朝AIニュースを収集し、フリーランスデザイナーに刺さるネタだけ atoms.csv に追加する。

## 実行手順
1. 直近24〜48時間の AI 関連ニュースを国内外から収集（WebSearch 必須）
2. 各ネタを以下の基準でスコアリング（5点満点）
   - 読者のタスクに直接影響するか（+2）
   - 一次情報か／公式発表か（+1）
   - 具体的な数値・日付があるか（+1）
   - 他媒体で既に飽和していないか（+1）
3. スコア 3 以上のみ atoms.csv に `status=new` で追記
4. id は `aYYYYMMDD-NN` 形式

## 出力
- atoms.csv への追記
- 収集総数 / 採用数のサマリー
