# post-social

## 目的
note記事・インサイトをもとに X と Threads の投稿文を生成する。
各アルゴリズムの特性に合わせ、プラットフォームごとに別文を作る。

---

## X 投稿ルール（2026年アルゴリズム準拠）

### アルゴリズム特性
- 返信はいいねの **150倍** の重み → 返信を促す終わり方が最優先
- 投稿後 **30分以内** のエンゲージメント速度が最重要シグナル
- 外部リンクは本文禁止（リーチがほぼゼロになる）→ **1つ目のリプライに置く**
- テキストファーストが動画より30%優勢
- スレッド形式（1→2→3）が単発より伸びる

### フォーマット
```
[断言 or 逆張り or 数字で始まる1行]

[本文：2〜4行。具体的事実 or 体験談]

[問いかけ or 行動促進で締め]

#ハッシュタグ1 #ハッシュタグ2
```

### 制約
- 140字以内（リプライ誘導文含む）
- ハッシュタグ2個まで（本文末尾）
- 外部リンクは本文に入れない
- **1案のみ生成**。ターゲットの痛みに直撃する最良案を1本に絞る

---

## Threads 投稿ルール（2026年アルゴリズム準拠）

### アルゴリズム特性
- **画像必須**（テキストのみは最もリーチしない）
- 外部リンクは For You 圏外になる → 本文に入れない
- 投稿後 **30分以内** の返信数がFor You配信量を決める
- フォロワー外（For You）に届きやすい = 新規リーチ向き
- 会話・共感・質問形式が最も伸びる

### フォーマット
```
[共感・問いかけ・体験談で始まる1〜2行]

[本文：3〜5行。読者の日常に接続した内容]

[返信したくなる質問 or 「あなたはどう思う？」で締め]

#ハッシュタグ1 #ハッシュタグ2
```

### 制約
- 500字以内（Threadsの上限は500字）
- ハッシュタグ2個まで
- 外部リンクは入れない
- 画像が必要な旨をメモとして添付（「サムネイル画像を使用」など）
- **1案のみ生成**。返信したくなる問いかけで締める最良案を1本に絞る

---

## ターゲットの痛みポイント（投稿生成前に必ず参照）
- 仕事を全部1人でこなして疲弊している
- AIに興味はあるが何から始めればいいか分からない
- クライアントから「AIで早くなるでしょ」と値切られそうで怖い
- デザイナーとしての価値がAIに奪われるかもしれないという不安
- 忙しすぎて学ぶ時間が取れない

投稿はこの痛みに1つ刺さるものを選び、共感→事実→次の行動 の順で組み立てる。

## 出力フォーマット

```
## X
（本文140字以内）
#ハッシュタグ1 #ハッシュタグ2
※リプライ用リンク: [URL]

## Threads
（本文）
#ハッシュタグ1 #ハッシュタグ2
📎 画像: サムネイル画像を使用
```
