# Adobeが「作る人」から「指揮する人」へ舵を切った日

2026年4月15日、Adobeは Firefly の新しい顔として「Creative Agent」を発表しました。これは単なる新機能ではなく、クリエイターの職能そのものを問い直す動きです。この記事を読むと、Photoshop や Illustrator を日常的に開いてきたデザイナーが、来月から何を準備すればいいのかが見えてきます。

## 何が発表されたのか

Creative Agent は、Firefly の中に統合された会話型AIアシスタントです。ユーザーは「このキービジュアルの色味を全部秋っぽく変えて、インスタ用と縦長バナー用に書き出して」といった自然言語の指示を出すだけで、Photoshop・Illustrator・Premiere・Lightroom・Express・Firefly を横断した複数ステップの作業をエージェントが代行します。

ポイントは3つ。

- 対話ひとつで複数アプリをまたぐ
- 途中のステップを人間が止めて手直しできる（全自動ではない）
- 最終成果物の「方向性」を人間が決める前提で設計されている

Adobe はこの変化を「クリエイティブディレクターの時代」と呼んでいます。作業を手で積み上げる人から、意図を言語化して指揮する人へ、というメッセージです。

## なぜフリーランスデザイナーに関係あるのか

結論から言うと、価格の根拠が変わります。

これまでデザイナーの工数は「何時間かけて作ったか」でおおむね説明できました。バナー1枚のレタッチに3時間、LP1ページに12時間、という感覚です。Creative Agent が普及すると、その3時間が30分に圧縮される仕事が相当数出てきます。クライアントが同じ時間単価で発注し続ける保証はありません。

一方で、圧縮されない仕事もあります。

- 「何を作るか」を決める要件整理
- ブランド全体を通貫する判断
- エージェントの出力が“惜しい”ときに一発で直せる手の速さ

要するに、上流と仕上げの両端が残り、中間工程が薄くなる。これは数年前から言われてきた話ですが、Creative Agent はそれを Photoshop の中で強制的に体験させる設計になっています。

## 明日からできる3つの準備

大きな買い物や契約変更は要りません。明日からできることを3つに絞ります。

1. 自分の仕事を「指示文」で書き直してみる。直近3件の納品物を、Creative Agent に投げるならどう書くか、日本語のプロンプト案として手元に書き出す。自分のブランド感覚が言語化できていない箇所が一発で浮き出ます。
2. 工程を2列で記録する。「手で動かした時間」と「判断に使った時間」を分けて記録する。判断時間こそがあなたの単価の根拠になるので、それを見える化する癖をつける。
3. クライアントへの説明文を用意する。「AI活用で早くなった分、値下げする」ではなく「AI活用を前提に、ディレクション含めて一括で引き受ける」という建て付けのメールテンプレをひとつ書いておく。問い合わせが来てから考えると遅いので、先回りします。

## 私の視点

正直なところ、Creative Agent は私自身にとっても脅威と追い風の両方です。

脅威は、単発の制作案件が「自分でやる人」に吸収されやすくなること。追い風は、ディレクションや要件整理を言語で回せる人に仕事が集中しやすくなること。後者にまわるには、「良し悪しを一言で言える」訓練を重ねるしかありません。自分が作ったものを自分で採点する習慣は、エージェント時代にそのまま武器になります。

このニュースは「AIに仕事を奪われる」話ではなく、「デザイナーの仕事が、手を動かすことから意図を伝えることへ、正式に引っ越した」という話です。引っ越し通知はもう届いています。

## 今週のあなたの一歩
