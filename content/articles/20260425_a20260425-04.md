現時点（2026年4月25日）では、Pro $20プランでもClaude Codeは引き続き使えています。ただ、同じことがまたいつ起きてもおかしくないので、このタイミングで、自分が今払っているAIツールを全部並べて見直しました。

## なぜ「増やさない」を決めたか

AIツールは、使い始めるのが簡単です。無料トライアルで試して、便利だから課金して、でも他のツールでも同じことができて。気づいたら似たような機能に月3本課金していた、というのがよくあるパターンです。

私が問題だと思ったのは、ツールが増えるほど「何をどれで使うか」の判断コストが増えることです。起動するツールを迷う。同じ作業を違うツールで二度やってしまう。これは効率化どころか、摩擦を増やしているだけです。

「新しいサブスクを増やすとき、今あるどれかを手放せるか」を判断基準にしました。

書き出してみると、用途が重なっている箇所と、契約上すぐ動かせない箇所が見えました。

## 「残す」「更新時に見直す」を分けた基準

私が使った判断軸は3つです。

**1. 代替できないか**

Claude Codeはここに当たります。他のツールで再現できない自動化が仕事の中心にあるので外せません。

Adobe CCは少し違います。本音を言うと、代替したい。

- Webデザイン・UI系：Figmaの無料枠で案件が回っている
- SNS・販促物系：Canvaで固定案件がある
- 印刷・DTP系：IllustratorとPhotoshopが必須で、ここだけAdobe CCが残っている

つまり移行は、気づかないうちに半分進んでいました。

AffinityなどAdobeの代替ソフトも調べました。でも踏み切れなかった理由が3つあります。

- **慣れすぎている**：ショートカットも、パネルの位置も無意識で動いている。慣れ直す時間コストは料金の差額より高くつく可能性がある
- **クライアントの縛り**：「Aiデータで納品してほしい」と言われることがある。形式の縛りがある以上、完全には離れられない
- **職員割が使えている**：定価だったら、仕事の方針ごと見直していた。割引が決断を先送りにしています。同じ状況のデザイナーは多いはず

**2. 収益に直結しているか**
Canva Proは公式クリエイターとして収益が発生しており、固定のデザイン案件も受けています。課金額より収益が上であれば、それは固定費ではなく投資です。外す理由がありません。

**3. 今すぐ動かせるか**
ここで壁になったのがSkyworkです。お試しのつもりで契約したら、年間プランになっていました。完全に確認ミスです。Claude ProやClaude Designと用途が重なるので整理したいのですが、期間が残っているのでフル稼働させている。笑えない笑い話として、棚卸し表に堂々と居座っています。
Skyworkは期日満了でさよならする予定。

## Claude Designへの集約を考えている理由

Anthropicが2026年4月に発表したClaude Designは、テキストからプロトタイプ・スライド・1枚物を生成し、Canvaへエクスポートできるツールです。Claude Pro契約者は追加費用なしで使えます（現在は研究プレビュー）。

私がこれに注目している理由は、「新しいサブスクを増やさない」という方針と完全に一致するからです。リサーチ・文章生成・デザイン試作をClaude Pro一本に集約できるなら、Skyworkの更新をやめる根拠がより強くなります。

デザイナー視点で見ると、できることが面白い。

- テキスト指示だけでプロトタイプ・スライド・1枚物を生成
- Canvaへそのままエクスポートして編集できる
- クライアントへの初稿提案やLP・提案資料の叩き台として使える可能性がある

Canva Proをすでに使っている私には、ワークフローとの相性もよさそうです。まだ実務では試せていませんが、触るのが楽しみなツールです。Claude Proの中に答えがあるなら、新しいサブスクを増やす前にここを使い切りたい。

## 棚卸しの手順（そのまま使えます）

同じことをやってみたい方向けに、私がやった順番を残しておきます。

1. 今払っているAIツールを全部書き出す
2. それぞれの「実際の使用頻度」と「主な用途」を1行で書く
3. 用途が重なっているものをマークする
4. 重なっているうちの「どちらか1つだけ残すとしたら？」を決める
5. 手放す候補を1ヶ月だけ使わずに仕事してみる

5番が一番大事です。「なくなると困る」かどうかは、使わない期間を作って初めて分かります。

ステップ2〜4は、Claudeに手伝ってもらうと10分以内に終わります。以下をそのままコピーして使ってください。

**ステップ2：用途と頻度の整理**

```
このツールの使い方を整理して。
ツール名：
使う場面：
週に使う回数：
```

**ステップ3：重複チェック**

```
このリストで用途が重なってるものを教えて。どちらか1つ残すならどっち？
（ツール名と用途を貼り付ける）
```

**ステップ4：手放す判断**

```
〇〇を解約するか迷ってる。
使う場面：
代わりに使えそうなツール：
残すべきか手放すべきか判断材料を出して。
```

このプロンプトは、棚卸しをしながら契約中のAIツールをフル活用するためのものです。せっかく課金しているなら、そのツールで何ができるかを自分で把握しておく必要があります。宝の持ち腐れにしないために。スキマ時間10分でできます。

## 私（ciro）の視点

整理してみて気づいたのは、「合理的に残している」と「惰性で残っている」は見た目が同じということです。

Adobe CCは割引がなければ手放していた。Skyworkは確認ミスで縛られている。どちらも「主体的に選んだ」ではなく「状況に残されている」ツールです。

1人で仕事を回しているデザイナーにとって、固定費の判断は事業の方針判断と地続きです。Adobeを手放すとき、私はAdobeだけでなく、今後どんな仕事を受けるかも一緒に決めることになる。だから踏み切れない。そのリアルを書いておきたかった。

「本当に必要か」を考えるのに、コストだけを見ていても答えは出ません。そのツールを使い続けることで、自分が何者であり続けるかを決めている側面があります。

## 最後に

今の私のスタックは、整理した結果こうなっています。

- Claude Pro：残す（代替不可）
- Adobe CC：残す（職員割がある間は。定価になったら見直す）
- Canva Pro：残す（収益が出ている）
- Skywork：更新しない予定（契約期間中は使い倒す。Claude Designで代替できるか検証中）

ツールを選ぶことは、働き方を選ぶことだと思っています。Adobeを手放せないのは、コストの問題だけじゃない。そのツールを使い続けることで、どんな仕事を受け続けるかを決めている。棚卸しは、固定費の整理じゃなくて、自分のキャリアの棚卸しでもありました。
